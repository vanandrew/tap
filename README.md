# tap
An indexing database for the ABCD dataset

### Getting Started

#### Development

##### Prerequisites
**tap** uses `python3.6+`.

Simply install the packages in `requirements.txt`.

```
pip install -r requirements.txt
```
####
Run the test server by running `manage.py` under the `tap` folder.

```
./manage.py runserver
```
